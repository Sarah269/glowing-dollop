 # Outliers

This project looks at four methods of identifying outliers using Python or SQL.

Tools:
*  Oracle LiveSQL
*  Python
*  Jupyter notebook

Methods Explored:
*  Sorting
*  Boxplot
*  Interquantile Range (IQR)
*  Z-score
